# Fallout Photo Booth

## A ESP32 Based photo booth made for the HackClub Fallout Hackathon by the Soup Soup Soup Sahur team

## Description

**Why we made this:** The theme for the hackathon was that the world is ending and we need to create a cool project. We made the photo booth, because we want this event to be remembered by aliens if they come to earth after everyone died.

**How to use:** First you need to press the photo button, wait for the countdown and pose for the picture, wait for the printer to print the image and donwload the photo via the QR code on the screen if you want. Thats it!

**How to build:**: Get the XIAO ESP32 S3 SENSE, UEDX80480043E-WB-A Display, A XP-203A thermal Printer. Flash them with their firmware (camera-module for the Xiao, Main-module for the display). Wire them like written in the firmware config files and place the filters on the SD in a folder called "Filters".
